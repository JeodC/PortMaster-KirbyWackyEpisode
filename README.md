## Default Gameplay Controls
| Button | Action |
|--|--|
|START|Pause|
|SELECT|Quit|
|D-PAD Right|Superstar Bullet|
|D-PAD Down|Swallow / Copy Ability|
|EAST|Cancel / Inhale|
|SOUTH|Confirm / Jump / Float|
|L2|Swap Ability|
|R2|Block|

\* Controls are configurable in the options menu
